# MVP Scope Guard

Cartwise MVP must keep only these active features:
1. Landing pública
2. Login demo
3. Dashboard mensual
4. Catálogo / búsqueda de productos
5. Compra pendiente
6. Comparación de supermercados
7. Plan recomendado
8. Historial de compras
9. Listas guardadas simples
10. Despensa simulada

Do not add:
- OCR
- real authentication
- payments or subscriptions
- maps or routes
- predictive pricing
- AI recommendations
- family mode
- collaborative lists
- community features
- real-time price promises
- UI scraping controls

Before adding any feature, ask:
Does this directly support searching products, preparing a pending purchase, comparing supermarkets, recommending a plan, confirming/saving a purchase, tracking monthly food spending, or managing the simulated pantry?

If not, leave it out or mark it as post-MVP.
