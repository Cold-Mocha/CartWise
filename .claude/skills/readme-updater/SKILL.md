# README Updater for Cartwise

README must describe the real MVP, not the full future vision.

Include:
- What Cartwise is.
- Current MVP scope.
- Active supermarkets.
- Coming soon supermarkets.
- What the MVP does not do.
- Main user flow.
- Data transparency: snapshot-based prices, not real-time.
- How to run:
  - npm install
  - npm run dev:full
  - npm run build
  - npm run lint
- Notes for future AI agents/developers.
- Deprecated/out-of-scope files if present.

Do not claim:
- OCR is active unless implemented.
- login is real.
- payments are active.
- prices are real-time.
- Tottus/Líder are active.
