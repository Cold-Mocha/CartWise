# UX Copy Spanish for Cartwise

Write concise Spanish UI text.

Rules:
- Prefer clear verbs: “Agregar”, “Comparar”, “Guardar”, “Confirmar compra”.
- Use “compra pendiente”, not “canasta activa”.
- Use “diferencias destacadas” for price gaps.
- Use “ofertas temporales” only for actual offers.
- Use “precios referenciales según último snapshot disponible”.
- Avoid exaggerated marketing.
- Do not promise real-time prices.
- Empty states should guide action.

Examples:
- “No tienes una compra pendiente. Busca productos o repite una lista guardada.”
- “Este producto ya está registrado en tu despensa.”
- “La recomendación prioriza cobertura y luego precio.”
- “Oferta detectada en el último snapshot.”
