# Iconos clave de Cartwise

Iconos descargados de **Lucide** (https://lucide.dev · licencia ISC, uso libre).
Cada uno mapea a un elemento clave del flujo del proyecto (comparar canastas de
supermercado). Úsalos como `<img src="/icons/archivo.svg" .../>` o como referencia
para el componente `lucide-react` equivalente.

| Elemento del proyecto | Archivo | Uso sugerido |
|---|---|---|
| Canasta de compra | `shopping-basket.svg` | Canasta, nav "Comparar", estado vacío |
| Carrito de compra | `shopping-cart.svg` | Alternativa a la canasta |
| Ahorro | `piggy-bank.svg` | Ahorro estimado, recomendación |
| Presupuesto / billetera | `wallet.svg` | Total, gasto estimado |
| Comparar (balanza) | `scale.svg` | Comparación entre tiendas, "comparable por unidad" |
| Supermercado / tienda | `store.svg` | Tarjetas de tienda, "tiendas conectadas" |
| Precio / etiqueta | `tag.svg` | Precio del producto, ofertas |
| Boleta / total | `receipt.svg` | Total por tienda, plan guardado |
| Buscar | `search.svg` | Buscador de productos |
| Producto / paquete | `package.svg` | Ítem de canasta, resultado de búsqueda |
| Mejor opción | `trophy.svg` | Sello "Recomendada" |
| Comparable exacto (verificado) | `badge-check.svg` | Match "Exacto por código de barras" |
| Precio más bajo / ahorro | `trending-down.svg` | Diferencias destacadas, mejor precio |
| Historial | `history.svg` | Nav "Historial", planes guardados |
| Frescura del dato / fecha | `calendar-clock.svg` | "Precios actualizados al…" |
| Comuna / zona | `map-pin.svg` | Preferencia de comuna |
| Filtros | `funnel.svg` | Filtros de búsqueda |
| Preferencias / ajustes | `settings.svg` | Vista Perfil → Preferencias |
| Ver en tienda | `external-link.svg` | Enlace a la oferta en la tienda |
| Disponible | `circle-check.svg` | Estado de línea: disponible |
| Agotado / sin precio | `circle-x.svg` | Estado de línea: agotado / sin precio |
